## Jupyter

### Start

Launch Jupyter over the Anaconda-UI or (preferred) over the CLI:

```bash
jupyter notebook
````

### Shortcuts

* ESC + m: Change to Markdown-Cell
* ESC + y: Change to Code-Cell
* ESC + b: create cell below
* CRTL + Enter: run cell
* Enter: enter edit mode

```bash
# line break
<br>
```
